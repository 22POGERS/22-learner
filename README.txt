# Интерактивный учебник для дистанционного обучения учеников школы №22 г.Ставрополя

Представляет собой web-сайт, состоящий из тестов по разным предметам. Сами задания редактируются и добавляются в ветке quiz.




Участники: 
Команда разработчков-практикантов (Кузнецов Артём, Кирпаль Екатерина); 
Главный бухгалтер (Афанасенко Е.Г.); 
Заместитель директора по УВР (Соловьёва А.А.); 
персонал школы (учителя)

Каждый из участников должен решать следующие задачи:
1.	Спроектировать программный продукт.  
2.	Разработать web-сайт. Ответсвенными за это задание была назначена команда разработчиков-практикантов.
3.	Протестровать программу. Ответсвенными за тестрование была назначена команда разработчиков-практикантов.
4.	Расчёт затрат. Ответственным за выполнение этой задачи был поставлен главный бухглатер школы.
5.	Внедрение программного продукта. Заместитель директора по учебно-воспитательной работе, персонал школы.

Для создания идеи о реализации онлайн-тестирования, был использован самый простой и популярный метод генерации идей – мозговой штурм. Всеми участниками были выдвинуты идеи по улучшению учебного процесса в МБОУ СОШ №22, после чего все смешав, на выходе получили web-сайт с платформой для тестирования учащихся.

Cписок требований:
1.	Программный продукт должен быть реализован по окончанию ПП.03
2.	Должны выполнятся все основные свойства требований
3.	Дизайн, выбор технологий реализации web-сайта выбирает команда разработчиков-практикантов.
4.	Доступ к измененям банка заданий и исходного кода должен быть только у отвестсвенного за редактирование Git-репозитория.
5.	Web-сайт должен поддерживаться на компьютерах организации-заказчика.
